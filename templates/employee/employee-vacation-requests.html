Employee Vacation Requests - Content Area

This page is dedicated to allow an employee to request a vacation. There will be some sort of form field that asks details of the vacation like dates, duration, cost to organization, etc. This request, as well as other previous ones will be displayed in a table , with columns that contain the details of the vacation request, and the status/comments on the request from both manager and admin level.
